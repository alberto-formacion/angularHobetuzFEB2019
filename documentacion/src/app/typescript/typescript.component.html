Caracteristicas de TypeScript

Instalar

npm install -g typescript

Tipado de variables, métodos

En TypeScript podemos tipar nuestras variables:
  • let user: string
  • let edad: number
  • let alumnos: Array<Alumnos>

También en los métodos el tipo de retorno y los parametros:

function suma(numero1: number, numero2: number): number {...}

Otra capacidad de TypeScript son los valores por defecto:

function suma(numero1: number = 0, numero2: number = 0): number {...}

Interfaces

Las interfaces se utilizan para definir un patrón

interface sumergibleInterface { tiempoMaxBajoElAgua: number; profundidadMaxima: number; repelerAgua(): void; }

Clases

Las clases es donde declararemos nuestras variables y métodos, se podría decir que son tipos de datos creados por nosotros.

class relojSumergible implements sumergibleInterface { tiempoMaxBajoElAgua = 1; profundidadMaxima = 10; repelerAgua() { console.log('El agua me resbala'); } }

Compilado

Para compilar un archivo:

tsc prueba.ts

Para compilar un proyecto:

  • Crear tsconfig.json: tsc --init
  • Compilar el proyecto: tsc -p ./
