Routing

El Routing lo utilizamos para crear nuestras SPA y poder navegar entre componentes.

Para esto tenemos que hacer:

  1. En el index.html añadir
  2. Añadir las rutas en la variable routes del AppRoutingModule
    • const routes: Routes = [
        { path: 'heroes', component: HeroesComponent },
        { path: 'not-found', component: NotFoundComponent },
        { path: '**', redirectTo: 'not-found' }
      ];
