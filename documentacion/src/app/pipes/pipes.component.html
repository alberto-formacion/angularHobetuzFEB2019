Pipes

Los pipes nos permiten modificar los datos que exponemos en las vistas, es decir formatear la información que presentamos al usuario.

Angular nos ofrece unos pipes preconstruidos (Built-in). Algunos de estos son:

  • DecimalPipe {{ value_expression | number [ : digitsInfo [ : locale ] ] }}
  • UpperCasePipe / LowerCasePipe {{ value_expression | uppercase }}
  • CurrencyPipe {{ value_expression | currency [ : currencyCode [ : display [ : digitsInfo [ : locale ] ] ] ] }}
  • DatePipe {{ value_expression | date [ : format [ : timezone [ : locale ] ] ] }}

Pipes de Usuario

Angular nos permite la creación de nuestros propios Pipes

Para esto utilizaremos el decorador @Pipe y también haremos que nuestra clase pipe implemente la interfaz PipeTransform lo que hace que tengamos que implementar el método transform.

El método transform recibe por parametro el elemento que queremos modificar y los parametros extras que necesitemos para la correcta transformación y debemos devolver lo que queremos mostrar transformado.

A estos Pipes se les llama exactamente igual que a los Built-in pero con el name que le hayamos puesto.
