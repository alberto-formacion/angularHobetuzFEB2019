Componentes

Los componentes son como el bloque básico de construcción en una aplicación Angular.

Estan compuestos por:

  • Selector
  • HTML (template)
  • Estilos (style)

Los componentes se decoran con @Component(), este decorador recibe un array de valores:

  • selector el nombre que utilizaremos donde queremos utilizar este componente.
  • templateUrl fichero donde tenemos nuestro código HTML.
  • template código HTML incrustado en el componente directamente.
  • styleUrls fichero donde tenemos los estilos del componente.
  • styles código CSS directamente incrustado en nuestro componente.

Estas son las mas comunes pero hay mas opciones...

Comunicación entre Componentes

Los componentes se pueden comunicar entre ellos pero siempre de padre a hijo o de hijo a padre.

Comunicación Padre-Hijo

En el padre (contenedor):

  • <app-hijo [variable]="valorParaEnviar"></app-hijo>

En el hijo (presentador):

  • @Input() variable: any;

Comunicación Hijo-Padre

En el hijo (presentador):

  • En el componente:
    • @Output() public emitter = new EventEmitter<any>();
    • enviar(valorParaEnviar: any): void { this.emitter.emit(valorParaEnviar); }
  • En el template:
    • <button (click)="enviar(valorParaEnviar)">

En el padre (contenedor):

  • <app-hijo (emitter)="metodoParaTratarComunicacion($event)"></app-hijo>
