Routing

El Routing lo utilizamos para crear nuestras SPA y poder navegar entre componentes.

Para esto tenemos que hacer:

  1. En el index.html es obligatorio añadir <base href="/"> para que Angular pueda hacer el routing.
  2. En AppRoutingModule definimos el módulo (@NgModule).
  3. Añadir las rutas en la variable routes del AppRoutingModule.
    • const routes: Routes = [
        { path: 'heroes', component: HeroesComponent },
        { path: 'not-found', component: NotFoundComponent },
        { path: '**', redirectTo: 'not-found' }
      ];
  4. Crear un menu en el que las rutas routerLink="/inicio" en lugar de los href.
  5. Para hacer que la opción del menú se marque según la página en la que estamos hay que poner routerLinkActive="active", donde active es la clase CSS que queremos que se le ponga a la opción del menú cuando este activo.
